# Color

The design system color palette consisting of a collection of color scales in a
range of contrast ratios from the background color.

## Color Palette

Here's the list of variables available for customization:

<ColorPalette>
  {Object.entries(colors.Foundations.Color).map(
  ([paletteName, paletteColors]) => 
    <ColorItem
      key={paletteName}
      title={paletteName}
      colors={Object.fromEntries(
        Object.entries(paletteColors).map(([colorName, colorInfo]) => [
          `${colorName}`, colorInfo.value,
          `${colorName}`, colorInfo.value,
          `${colorName}`, colorInfo.value
        ])
      )}
    />
)}
</ColorPalette>
